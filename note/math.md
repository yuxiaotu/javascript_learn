## 02. 取整
`round()` 方法返回最接近指定值的整数。

```js
Math.round(4.5) // 5
Math.round(4.4) // 4
```

`ceil()` 方法向上取整。

```js
Math.ceil(4.2) // 5
Math.ceil(-4.2) // -4
```

`floor()` 方法向下取整。

```js
Math.floor(4.9) // 4
Math.floor(-4.2) // -5
```

`trunc()` 方法取整数部分。

```js
Math.trunc(4.9) // 4
Math.trunc(-4.2) // -4
```
