# null 和 undefined 的区别

# 1. undefined
`undefined` 表示的是「为定义的值」，表示一种原始状态。

- 声明一个变量，但是没有赋值
- 访问对象上不存在的属性或者未定义的变量
- 函数定义了形参，但没有传递实参


# 2. null
`null` 表示「空值」，表示一个对象被人为的重置为空对象，而非一个变量最原始的状态 。

- 当定义的变量最后会保存对象，那就将它赋值为 null。
- 当一个数据不在需要时，赋值为 null，来释放其他引用。

```js
var data = null;
typeof data // "object"
```
